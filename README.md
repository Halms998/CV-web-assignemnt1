# CV Website

## Project Information
This is a personal CV (Curriculum Vitae) website designed to showcase professional experience, skills, and qualifications. The website features an interactive and responsive design with a clean user interface.

## Technologies Used
- **HTML5** - Markup structure
- **CSS3** - Styling and layout
- **JavaScript** - Interactive functionality and toggle features

## Features
- **Single Page Layout** - All content organized on one page for easy scrolling and navigation
- **Hover Animations** - Smooth and interactive hover effects on elements for enhanced user experience
- **Light/Dark Mode** - Toggle button to switch between light and dark themes
- **Download CV Option** - Easy one-click download functionality for the CV
- **Responsive Design** - Works seamlessly on all devices
- **Professional Styling** - Clean and modern design

## Live Link
https://haleemah-cv.vercel.app/

## How to Use
1. Open `index.html` in your web browser
2. Scroll through the single page to view all CV sections
3. Hover over elements to see interactive animations
4. Click the toggle button to switch between light and dark modes
5. Use the download button to save the CV as a file
